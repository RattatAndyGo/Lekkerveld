Cookie usage on Lekkerveld.be

Below is a summary of all cookies used on Lekkerveld, and their purpose.

  • username
    Used for users to identify themselves, and to check on their friends' hunts.
  • idToken
    The main reason I even store data on server instead of all locally, is so users can access the same data on multiple devices. As users can check out other users' hunts using their username, another cookie had to serve for (somewhat) secure identification. idToken cannot easily be shared accidentally, so it is used in all server requests requiring identification.
    The server does not use the username cookie to identify users in requests, rather it uses idToken.
  • board
    This stores the inputs of a bingo card.
