This page is not meant to be frequently visited, but rather serves as a tutorial to use this feature. The frontend above to get random images is purely for asthetics and experimenting.
The idea is to have a keyboard shortcut that, when pressed, automatically loads a random image of a particular theme into your clipboard. You can then immediately use ctrl + v to paste the image anywhere. Currently, only angry emoji themed images are accessible.
The process behind this is relatively simple: When the shortcut is pressed, a shell script is activated, which retrieves the image from www.lekkerveld.be/get-random-image and places it in your clipboard. To implement this on your own device, follow the instructions below.

Linux

  1. install xclip
    This should be already installed, which can be checked by typing "which xclip" in a terminal. If the output is similar to "/usr/bin/xclip", it is installed, else go ahead and install it (see the internet for guides)
  2. set up the shell script
    Make a file named shellscript.sh (any name is fine, just have it end in .sh) and add the following as content:
    #!/bin/bash
    curl https://www.lekkerveld.be/get-random-image | xclip -selection clipboard -target image/png -i
    Make sure you know where you save this file.
  3. add the keyboard shortcut
    This is different for every Linux distro, but it should be similar to Settings > keyboard > keyboard shortcuts > custom shortcuts. Create a new shortcut, and give it any name and key combination. The command should be "sh /home/username/path/to/script/shellscript.sh", with username being swapped with your own username, path/to/script being the path to the script, and shellscript.sh should be the actual name you gave the script.
  4. test it Everything should be configured now. Use the shortcut and try to paste the image somewhere (I suggest testing it wherever you are planning on using this for, I chose Discord for this very reason). If something doesn't work, you can always manually run the shell script in a terminal and inspect the error message to try and solve any issues.
