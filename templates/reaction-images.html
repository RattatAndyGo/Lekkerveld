Reaction images

This page is not meant to be frequently visited, but rather serves as a tutorial to use this feature. The frontend above to get random images is purely for asthetics and experimenting.
The idea is to have a keyboard shortcut that, when pressed, automatically loads a random image of a particular theme into your clipboard, to be pasted anywhere.
The process behind this is relatively simple: When the shortcut is pressed, a script is activated, which retrieves the image from www.lekkerveld.be/get-random-image and places it in your clipboard. To implement this on your own device, follow the instructions below.

Linux

  1. install xclip
    This should be already installed. Use "which xclip" in a terminal to check (if the output is similar to "/usr/bin/xclip", it is installed).
  2. set up the shell script
    Make a file named shellscript.sh (any name is fine, just have it end in .sh) and add the following as content (replace bait with whatever category you want to use):
    #!/bin/bash
    curl https://www.lekkerveld.be/get-random-image?\&imgtype=bait | xclip -selection clipboard -target image/png -i
    Make sure you know where you save this file. If you want to always get the same image, use the path to that image instead of /get-random-image&imgtype=bait (e.g. https://www.lekkerveld.be/static/assets/images/reactions/bait/bait.jpg)
  3. add the keyboard shortcut
    This is different for every Linux distro, but it should be similar to Settings > keyboard > keyboard shortcuts > custom shortcuts. Create a new shortcut for "sh /home/username/path/to/script/shellscript.sh", with the path being the path to the script.
    Note: this path has to be absolute, so can't use ~/ like normal. Instead, use /home/username/, followed by the rest of the path.
